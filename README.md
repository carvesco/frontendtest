### Frontend for the technical test

This is the frontend for the technical test developed using typescript with React and the MaterialUi component library

## 1. clone the repository
```bash
git clone https://github.com/carvesco/frontendtest.git
```

## 2.Install the dependencies
```bash
npm install
```
## 3.Start the app
```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

![image](https://user-images.githubusercontent.com/20958791/204160468-216472c1-18f5-4398-aade-3eeee351cc8e.png)

You need to have the backend server running in order to acces the app functionalities.
